## 概述

数据探针是 data-workbench 提供的一种数据标注和捕获工具，允许用户在绘图区域点击添加探针标记，记录点击位置的数据并在表格界面显示。探针功能特别适用于需要精确记录和比较特定数据点的场景。

## 探针类型

### 垂直探针（Vertical Probe）

垂直探针是一条垂直线，横跨整个绘图区域的Y轴范围。它用于标记特定的X值位置。

**特点：**

- 显示为垂直虚线
- 在线的顶部或底部显示探针名称标签
- 捕获X轴坐标值
- 与所有穿过该垂直线的数据曲线相交

**适用场景：**

- 标记特定时间点的数据
- 比较不同曲线在同一X值处的Y值差异
- 标注实验中的关键时刻

### 水平探针（Horizontal Probe）

水平探针是一条水平线，横跨整个绘图区域的X轴范围。它用于标记特定的Y值位置。

**特点：**

- 显示为水平虚线
- 在线的左侧或右侧显示探针名称标签
- 捕获Y轴坐标值
- 与所有穿过该水平线的数据曲线相交

**适用场景：**

- 标记阈值线或参考线
- 比较不同曲线在同一Y值处的X值差异
- 标注特定的数值水平

## 探针命名规则

### 默认命名

探针默认使用大写字母命名，从字母"A"开始递增：

- 第一个探针：A
- 第二个探针：B
- 第三个探针：C
- ...
- 第26个探针：Z
- 第27个及以后：AA, AB, AC, ...

### 自定义命名

用户可以为探针指定自定义名称，名称规则如下：

- 支持中英文字符
- 支持数字
- 不建议使用特殊符号
- 名称长度建议不超过20个字符

## 探针数据捕获

### 数据获取方式

探针提供以下数据获取接口：

| 接口 | 说明 | 返回值 |
|------|------|--------|
| `xValue()` | 获取探针的X轴坐标值 | `double` |
| `yValue()` | 获取探针的Y轴坐标值 | `double` |
| `value()` | 获取探针的坐标点 | `QPointF` |
| `name()` | 获取探针名称 | `QString` |
| `probeType()` | 获取探针类型 | `ProbeType` |

### 数据精度

探针捕获的数据精度取决于：

1. **坐标轴精度**：与绑定的坐标轴刻度精度一致
2. **数据类型**：支持整数和浮点数
3. **显示格式**：可通过格式化字符串控制显示精度

### 修改探针

1. **移动探针**：
   - 在属性面板中精确输入坐标值

2. **重命名探针**：
   - 在属性面板中修改名称

3. **修改探针样式**：
   - 线条颜色
   - 线条宽度
   - 线条样式（实线、虚线、点线）
   - 标签位置（顶部/底部、左侧/右侧）

## 探针数据表格

探针数据表格显示所有探针的信息：

| 列名 | 说明 |
|------|------|
| 名称 | 探针名称 |
| 类型 | 垂直/水平 |
| X值 | X轴坐标值 |
| Y值 | Y轴坐标值 |
| 创建时间 | 探针创建时间 |

### 表格功能

- **排序**：点击列标题进行排序
- **筛选**：支持按类型、名称筛选
- **导出**：支持导出为CSV、Excel格式
- **刷新**：实时更新探针数据

### 曲线追踪

探针支持曲线追踪功能：

- 显示探针位置处各曲线的数值

## 注意事项

1. **性能考虑**：
   - 大量探针可能影响绘图性能
   - 建议单个图表探针数量不超过50个

2. **坐标轴绑定**：
   - 探针绑定到创建时的坐标轴
   - 坐标轴范围变化时，探针位置自动调整

3. **数据更新**：
   - 数据更新后，探针位置保持不变
   - 如需更新探针数据，需手动刷新

4. **保存与加载**：
   - 探针信息随项目文件保存
   - 支持导出探针配置为独立文件
