# 使用指南

## 软件简介

DAWorkBench（数据工作台）是一个基于工作流驱动的ETL数据分析工具。软件的核心设计目标是：

1. **工作流**：通过有向图描述数据处理流程，解决重复性数据处理任务
2. **数据处理**：集成 Python pandas 的数据处理能力，提供交互式的数据清洗工具
3. **数据可视化**：实现交互式的数据可视化，支持拖拽调整图表元素，能生成论文级别的图片

## 系统要求

- **操作系统**：Windows
- **Qt 版本**：Qt 5.14+ 或 Qt 6.x
- **Python**：需要 Python 3.8+ 环境（程序会自动部署）

## 界面概览

DAWorkBench 的主界面由以下几个区域组成：

- **Ribbon 菜单栏**：位于窗口顶部，提供所有操作按钮，按功能分组（基于 SARibbon 库）
- **工作流编辑区**：位于中央，用于编辑工作流的有向图
- **数据操作区**：用于查看和操作 DataFrame 数据
- **图表编辑区**：用于创建和编辑图表
- **Dock 停靠面板**：位于四周，包含节点列表、属性面板、日志输出等辅助窗口

各区域均为可停靠面板（基于 Qt-Advanced-Docking-System），可以自由拖拽调整布局。

## 基本操作

### 工程文件

DAWorkBench 使用 `.dapro` 格式的工程文件，工程文件本质上是一个 ZIP 压缩包，包含工作流配置、数据文件和图表配置。

- **新建工程**：通过菜单新建空白工程
- **打开工程**：通过菜单或命令行参数打开已有工程文件
- **保存工程**：将当前工作状态保存为 `.dapro` 文件

### 数据导入

程序支持以下格式的数据导入：

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| CSV | `.csv` | 逗号分隔值文件 |
| Excel | `.xlsx` / `.xls` | Excel工作簿 |
| 文本文件 | `.txt` | 文本格式数据 |
| Pickle | `.pkl` | Python序列化数据 |

数据导入方式：

1. 通过 Ribbon 菜单中的导入功能
2. 通过命令行参数 `--import-data` 导入

### 工作流操作

工作流是 DAWorkBench 的核心功能，通过有向图描述数据处理流程：

1. 从左侧节点列表拖拽节点到工作流编辑区
2. 连接节点的输入/输出端口建立数据流
3. 配置节点参数
4. 执行工作流

### 数据可视化

创建图表的基本步骤：

1. 确保已导入数据
2. 选择要可视化的数据列
3. 通过 Ribbon 菜单选择图表类型
4. 在图表编辑区调整图表样式和布局

## 命令行参数

程序支持以下命令行参数，详见[程序命令参数](./command-line-arguments.md)。

| 参数 | 说明 |
|------|------|
| `--help` | 显示帮助信息 |
| `--version` | 显示版本号 |
| `[project]` | 打开工程文件 |
| `--import-data <path>` | 导入数据文件 |
